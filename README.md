# Marp Theme

## 概要
[Marp Web Editor](https://marpwebeditor.app/) を利用して、社内標準デザインのスライドを作成するためのテーマCSSおよびシステムファイルを管理するリポジトリです。

## ご利用の流れ

全体の手順は以下の通りです。社内AIの具体的な操作手順やプロンプトについては、**社内ポータルのマニュアル**をご参照ください。

1. **原稿の準備**

   スライド化したい文章（テキスト）をあらかじめ準備します。
2. **AIによるコード生成**

   社内AIに対し、所定のプロンプトとルールファイルを添付して原稿を送信し、Marpコード（Markdown）を生成させます。
3. **Marp Web Editorへの貼り付け【※重要※】**

   Marp Web Editorを開き、AIが生成したコードを適用します。初期状態ではサンプルが入力されているため、以下の手順で**必ず上書き・消去**を行ってください。
   * **Markdownタブ**：初期テキストをすべて消去し、生成されたコードを貼り付けます。
   * **CSSタブ**：初期CSSコードを**すべて削除し、空欄にしてください**。（※ここにコードが残っていると、社内テーマが正しく適用されません）
4. **テーマ調整・エクスポート**

   プレビュー画面でデザインを確認します。テーマを変更する場合は、以下の「登録テーマ一覧」を参考にコード先頭の指定箇所を書き換えてください。完成したらPDF、PPTX等でエクスポートします。

## 登録テーマ一覧
Marp Web Editorにコードを貼り付けた後、先頭にある `data-theme="corp-blue"` の部分を以下の名前に**手動で書き換える**ことで、即座にデザインが切り替わります。

* **`corp-blue`** : コーポレート標準デザイン（青ベース）※デフォルト
* **`corp-green`** : エコ・成長テーマ（緑ベース）

## 注意事項
本リポジトリは社内での運用を目的としています。利用しているSaaS（Marp Web Editor）の仕様変更等により、予告なく動作しなくなる場合があります。
