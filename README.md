# Marp Theme

## 概要
[Marp Web Editor](https://marpwebeditor.app/)を利用して、社内標準デザインのスライドを作成するためのテーマCSSおよびシステムファイルを管理するリポジトリです。

## 利用方法
本リポジトリのファイルは、社内のAIスライド生成フローから自動的に読み込まれます。スライドの具体的な作成手順やプロンプトの使い方については、**社内ポータル（または社内Wiki）のマニュアル**をご参照ください。

## 登録テーマ一覧
現在、以下のテーマが利用可能です（AIへの指示で切り替え可能です）。

* **`corp-blue`** : コーポレート標準デザイン（青ベース）
* **`corp-green`** : エコ・成長テーマ（緑ベース）

## 注意事項
本リポジトリは社内での運用を目的としています。利用しているSaaS（Marp Web Editor）の仕様変更等により、予告なく動作しなくなる場合があります。
